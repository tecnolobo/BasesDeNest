# Teslo Api

1. clonar proyecto
2. instalar paquetes
```
yarn install
```

3. renombrarb env.template a .env

4. cambiar las variables de entorno

5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar en modo desarrollo
```
yarn start:dev
```
